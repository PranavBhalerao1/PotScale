# PotScale — Design System

## Vibe
Modern AI startup aesthetic — think Linear, Arc browser, or Perplexity's app.
Clean, dark-mode-first, minimal chrome, generous whitespace, bold typography.

## Color palette
- Background: near-black (#0A0A0B) with subtle gradient overlays
- Primary accent: choose one vibrant color (e.g. electric green #00FF88,
  or electric blue #3B82F6) used sparingly for CTAs and highlights
- Surface/card color: dark gray (#1A1A1C) with soft borders (#2A2A2D)
- Text: white for primary, gray (#9CA3AF) for secondary

## Typography
- SF Pro Rounded or SF Pro Display for headings, bold weights
- Generous font sizes for headers (28-34pt), clear hierarchy
- Avoid default system gray text — use custom color tokens

## Components
- Buttons: rounded-full or large-radius rounded rects, filled with accent
  color for primary actions, subtle bordered style for secondary
- Cards: soft shadows, rounded corners (16-20pt radius), slight border glow
- Use blur/glassmorphism sparingly on nav bars or overlays for depth

## Motion
- Subtle spring animations on button taps and screen transitions
- Smooth fade/slide transitions between screens, nothing jarring

## Layout
- Lots of padding (20-24pt standard margins)
- Avoid cramming — one clear focal action per screen
